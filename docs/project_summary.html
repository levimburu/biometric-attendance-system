The Problem
Kenyan institutions currently rely on paper registers for attendance and identity verification. These registers are easily forged, manipulated, or lost — making it impossible to confirm that the right person was actually present. Exam impersonation, ghost workers, and falsified attendance cost institutions millions every year and undermine the integrity of our education system.
Our Solution
📱 Biometric verification
Fingerprint sensor identifies each person in real time. No cards, no signatures, no impersonation possible.
☁️ Cloud database
All records sync instantly to Supabase cloud. Management views live attendance from any device, anywhere.
💻 Admin dashboard
Full desktop and web app for managing students, lecturers, classes, units and attendance reports.
🛡️ Security built in
PIN lock, failed attempt lockout, timetable validation and real-time session monitoring included.
How It Works
1
Device powers on and displays a PIN lock screen. Operator enters PIN to unlock the system.
2
Session is started from the admin app — class, unit and lecturer are selected.
3
Students or staff place finger on the R305 biometric sensor. Identified in under 2 seconds.
4
Name and admission number display on OLED screen. Attendance logged to cloud instantly.
5
Management views real-time attendance reports on web dashboard from any browser.
